## Features

| Feature | Command |
|--------|--------|
| **Tag all members** | `.tagall` |
| **Admin-only commands** | Restricted to group admins |
| **Games** | Tic-Tac-Toe, etc. |
| **Text-to-Speech** | `.tts <text>` |
| **Sticker Creation** | `.sticker` (reply to image/video) |
| **Anti-Link Detection** | Auto-delete links |
| **Warn & Kick Members** | Admin control tools |

## About

**VAMPARINA V1** is a powerful, lightweight, and customizable WhatsApp bot built with **Baileys** for multi-device support. It helps group admins maintain order, engage members, and automate tasks — all without needing WhatsApp Web in a browser.

- **No QR scanning needed after first setup**  
- **Session-based authentication**  
- **Runs 24/7 on free hosting (Render, Replit, etc.)**
